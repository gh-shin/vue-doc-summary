## 1. 설치 방법
### CDN을 사용
    <script type="text/javascript" src="https://unpkg.com/vue@2.5.17/dist/vue.js"></script>
### NPM을 사용
    $ npm install vue
#### NPM 사용 시 [Vue CLI](https://cli.vuejs.org/)를 사용하여 프로젝트를 생성할 수 있다.
    $ npm install -g @vue/cli
    $ vue create my-project OR $ vue ui
